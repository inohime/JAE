
# Just Another Engine
![Example Image](example/assets/JAE_Test_Image.png)

## Currently In Active Development
  *   Non-production beta builds released frequently

`Dependencies:`
  *   SDL2 2.0.16+ (soon 2.0.18)
  *   SDL_Mixer 2.0.4
  *   SDL_TTF 2.0.15
  *   SDL_Image 2.0.5
  *   TMXLite (optional)
